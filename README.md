# ProTop
一个基于gank.io接口的干货阅读小应用,加入数据库和离线功能,能离线阅读,没网也可以阅读以前浏览过的文章.
主要技术:
RecycleView+BaseAdapterHelper+ViewPager实现数据呈现
Okhttp+Asynctask 实现网络数据请求异步加载
Sqlite实现数据存储
WebView 实现文章的阅读
